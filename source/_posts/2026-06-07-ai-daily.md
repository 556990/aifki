## Product Releases / Updates

**4. Thousand Token Wood v2: Building a Multi-Model Finance Drama Game with Small Models**
Source: HuggingFace

Thousand Token Wood v2 uses four different small models from various labs (gpt-oss-20b, MiniCPM3-4B, Nemotron-Mini-4B, and fine-tuned Qwen 0.5B) to power agents in a financial simulation game. The core finding is that heterogeneous service-layer friction lies in vLLM 0.22.1 requiring CUDA toolkit, not the models themselves. With a tolerant JSON parsing layer, adding models requires only a config line. Information isolation ensures insider flags don't leak into prompts, and scan tests verify no leakage. Memory uses emotional summary truncation to avoid drowning out. Fine-tuned 0.5B model achieved 0% self-trading and 100% valid quotes with zero truth firewall leakage. Small models are reliable format generators but unreliable reasoners, which can be compensated for through structure, prompting, and fine-tuning.
